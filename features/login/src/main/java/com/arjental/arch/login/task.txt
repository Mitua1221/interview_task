1. Проверить, был ли передан логин пользователя на экран
Error:
- если логин был передан, вставить логин пользователя в поле логина
- если пользователя не существует на бекенде, то показывать ошибку, что его не существует
Success:
- Если пользователь существует, загрузить картинку и загрузить пасскод поьзователя

2. Если пользователь будет изменять логин, перезагружать даные о картинке и пасскоде

3. По нажатию на кнопку входа проверять совпадение пасскодов
Error:
- если не совпадают, то показывать ошибку в поле ошибки
Success:
- Если пасскоды совпадают, то загружать данные пользователя и навигаировать на другой аккаунт
